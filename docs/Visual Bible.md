```ad-aia-conclusion
**A dreamy academic dashboard inspired by:**
- stage aesthetics
- magical archives
- idol imagery
- quiet late-night studying
- celestial motifs
```

- gamer HUD
- neon cyberpunk
- maximalist fantasy clutter

```ad-aia-gold
# 3. Visual Heirarchy
```

### A-tier components

Highest (visual) priority.

Examples:
- Hero banner (+ semi-transparent calendar)
- Current Arc / Project Report
- Focus card

#### Rules
- stronger gradients
- deeper glow
- larger typography
- more contrast
- richer illustrations

### B-tier components

Important information.

Examples:
- Deadlines
- Today's Schedule

The latter must be a section lower.
- Subjects

#### Rules
- medium contrast 
- softer borders (if any)
- structured layouts
- readable at glance

### C-tier components

Supportive information.

Examples:
- Recent Notes (most important)
- Growth Log
- Stats
- Quotes (footer)

#### Rules
- subtle
- airy
- low visual density
- less decorative noise

```ad-aia-gold
# 4. Layout Philosophy
```

### General Rules

```ad-aia-toc
dence top - airy middle - calm bottom
```
 *The page should gradually 'breathe'.*

### Layout Rhythm
#### Hero section

Wide and cinematic. Banner and Project section are divided by =={cyan} quick-add buttons==. 

#### Middle section

Asymmetrical composition.

=={red} Avoid:== 3 equal columns

=={yellow} Prefer:== large + narrow + floating accent

#### Lower section

Small modular blocks.
